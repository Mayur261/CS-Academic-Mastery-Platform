History of C

Martin Richards developed BCPL, which stands for Basic Combined Programming Language, in 1966.

Ken Thompson, who worked at Bell Labs, developed the B language in 1969. He was also the developer of the UNIX Operating System and created the first master-level chess program called Belle in 1980.

Dennis Ritchie is the developer of the C language. He did this work in 1972 at AT&T's LABs in the USA. He was also a co-developer of the UNIX Operating System.

Basic Structure of C Programming

  1. Documentation Section: In this section, we comment or note down information that is used in the future. Comments do not show in the output.
  2. Link Section: In this section, we include header files which contain pre-defined functions.
  3. Definition Section: In this section, we use symbolic Constants which are used in the program.
  4. Global Declaration Section: In this section, we declare global variables which are used anywhere in the program.
  5. Main Section: This Section is essential and is divided into two parts: (i) Declaration Part (ii) Executable Part
  6. Subprogram Section: In this section, we use different functions that are used throughout the program.

Variables

Variables are memory locations in a computer's memory used to store data. Each variable should be given a unique name called an identifier. Variable names are the symbolic representation of a memory location.

Example: int num; where int is the data type, and num is the variable name.

Rules for writing variable names in C:

  • Variable names can be composed of letters (both uppercase and lowercase), digits, and underscores '_' only.
  • The first letter of a variable should be either a letter or an underscore. Starting variable names with an underscore is discouraged.
  • There is no rule for the length of a variable, but the first 31 characters of a variable are discriminated by the compiler.
